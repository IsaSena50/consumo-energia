## 📝 Sobre o Projeto
Este é um sistema simples e eficiente desenvolvido em **Python** para ajudar usuários a estimarem o gasto de energia elétrica mensal de seus aparelhos domésticos. 

O objetivo é promover a conscientização sobre o consumo de energia e auxiliar no planejamento financeiro familiar.

---

## 🧮 Como o cálculo é feito?
O sistema utiliza a seguinte fórmula para chegar ao resultado:

$$consumoMensal = \frac{potencia \times horasDia \times 30}{1000}$$

> **Nota:** O resultado é exibido em **kWh/mês**.
